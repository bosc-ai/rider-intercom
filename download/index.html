Offline Guide

Seamless Offline P2P Setup

No routers or cell towers needed. Connect your devices in seconds using one of two offline local network methodologies.

01

Host Setup

The Host turns on their phone's Portable Wi-Fi Hotspot in settings (no mobile/cellular data is needed; leave data OFF to save battery).

02

Riders Connect

All other riders go to their Wi-Fi settings, locate the Host's hotspot name, and connect to the network.

03

Start Host

Host opens the app, enters their display name, and clicks START INTERCOM HOST. The Host's local IP address (typically 192.168.43.1) is displayed.

04

Riders Join

Other riders open their app, enter the Host's IP, and click JOIN INTERCOM SESSION. Microphones link instantly over local WebRTC!

01

Connect Router

Connect all phones to the same Wi-Fi router (e.g. at your home or basecamp). The router does not need an active DSL or fiber internet connection.

02

Identify IP

The designated Host clicks START INTERCOM HOST. The app automatically detects their router IP (e.g., 192.168.1.15) and displays it.

03

Input & Join

Other riders input the Host's exact IP address into their "Host IP Address" field and click JOIN INTERCOM SESSION. Your group audio session is immediately active.

For Developers

Build & Install in Seconds

Rider Intercom is open source and built using React Native. Follow these simple steps to build the APK and deploy it to your Android device directly from the codebase.

Make sure your Android device has **USB Debugging** enabled and is plugged into your system.

RiderIntercomNative_Terminal
# 1. Install project dependencies
npm install
# 2. Compile and launch on connected Android phone
npm run android
